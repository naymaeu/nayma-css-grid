Czym jest Nayma CSS Grid ?

Nayma CSS Grid to prosty, lekki (3kb gzip), responsywny framework CSS dzięki któremu stworzysz prosty układ 12 column (opartych o flexbox), nagłówki, listy, formularze oraz przyciski. Dodaliśmy również możliwośc dodania nawigacji górnej wraz z wysówanym menu bocznym w czystym kodzie CSS.

Dlaczego Nayma CSS grid ?

Nayma CSS Grid posiada wszystkie potrzebne elementy takie jak siatka kolumn, nagłówki, przyciski, formularze, tabele, menu i wiele innych. Zajmuje tylko (3kb gzip). Jest w pełni responsywny i posiada specjalne menu mobilne w czystym CSS. Oparte jest na nowoczestnych jednostkach takich jak REM, VH czy VW.

Siatka kolumn

Siatka Nayma CSS Grid korzysta z 12 kolumn, które ustawiamy na sztywno lub z pomocą flexbox. Szerokość kontenera ma maksymalnie 1260px, jednak można ją zmienić edytując jedną wartość w pliku css. Siatkę kolumn tworzymy w oparciu o wiersze o klasie .row. Same kolumny są oparte o flexbox więc wypełniają wiersz automatycznie. Korzystając z klasy `auto column` lub `auto columns` dodajemy kolumny korzystające z flexbox i wypełnia one wiersz automatycznie. Możemy jednak przypisać konkretną szerokość kolumnie od 1 do 12. Skorzystaliśmy z anglojęzycznych nazw klas by nasz kod był uniwersalny i każdy rozumiał terminologie. Poniżej przedstawiamy wszystkie klasy, których możemy używać do stworzenia siatki kolumn:

Przyciski

W Nayma CSS grid możemy korzystać z czterech różnych typów przycisków. Są to zwykłe odnośniki <a> z przypisaną klasą .button, elementy <button> oraz <input>. Wszystkie te elementy mogą być przyciskiem bez wypełnienia oraz z wypełnieniem.
