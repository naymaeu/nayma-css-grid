What is Nayma CSS Grid?

Nayma CSS Grid is dead simple, light (3kb gziped), responsive CSS framework/boilerplate whith twelve columns layout (whith use flexbox) to build simple websites. You will get styles for elements like headers, lists, forms buttons, tables and much more. Also you can use simple navbar for site navigation and mobile sidebar navigation with hamburger menu and slide in effect in pure CSS

Check Instalation Guide

Why Nayma CSS grid?

Nayma CSS Grid has very simple to learn syntax for build grid in CSS. Also have styled a lot of basic elements and is very light (3kb gziped). This bilerplate/framework is 100% responsive and have pure css nav elements for desktop and mobile. To build this framework we use REM, VH czy VW.

Check Our Documentation

Grid syntax

Nayma CSS Grid use 12 columns layout. You can use 100% width and cointener with max-width of 1260px. If you need widther container you can edit one line in css file. To start build your layout use firts div with class .row. Inside row you can add many columns you want and they use flexbox to fill row. You just need to use css class column. If you want to use specyfic width of column you can use this few simple to learn css class:
